BEL

Human readable, minimum syntax, language for capturing biological knowledge from scientific literature.

# Statement
tscript(p(HGNC:EGFR)) => g(EGID:207)

# Annotations
SET CellLine = "L6 cell line"
SET Species = "9606"
SET Evidence = "In this paper, we present a comprehensive pathway map of EGFR signaling and other related pathways."

# Citation
SET Citation = {"Other","Mol Sys Biol 2005 May msb4100014","Mol Sys Biol 2005 May msb4100014","","",""}
